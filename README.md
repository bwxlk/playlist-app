# Playlist App

A simple web app that lets users create playlists and add songs to them. 
Built with HTML, CSS, and JavaScript.

## Features
- Create playlists
- Click a playlist to view it
- Add songs (title, artist, link)
- See all songs in a playlist

## Future ideas
- Delete playlists
- Edit songs
- Save to localStorage
- User accounts & sharable links
